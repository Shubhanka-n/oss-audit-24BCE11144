## Scripts Description

### 1. System Identity Report
Displays system details such as OS, kernel, current user, uptime, date/time, and license information.

### 2. Package Inspector
Checks whether the chosen package is installed and displays package details.

### 3. Disk and Permission Auditor
Analyzes important system directories for existence, permissions, size, and disk usage.

### 4. Log File Analyzer
Counts keyword occurrences in a log file and shows the most recent matching lines.

### 5. Open Source Manifesto Generator
Generates a personal manifesto based on user input and saves it to a text file.

## How to Run Scripts

1. Open terminal
2. Navigate to project folder:
   cd oss-audit-24BCE11144

3. Give permission:
   chmod +x *.sh

4. Run scripts:
   ./script1_system_identity.sh
   ./script2_package_inspector.sh
   ./script3_disk_auditor.sh
   ./script4_log_analyzer.sh
